# RemoteStarterAPI

**RemoteStarterAPI** is a .NET Web API project that provides a basic template for managing users with Entity Framework Core and SQL Server. It includes:

- User management endpoints (CRUD operations)
- Database context setup using EF Core
- Swagger/OpenAPI documentation for easy API testing

## Features

- Simple API endpoints for creating, reading, updating, and deleting users
- Database integration with SQL Server
- Auto-generated Swagger documentation
- Ready for deployment or further development
